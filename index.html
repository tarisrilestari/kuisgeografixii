Hasil (nama siswa, skor, waktu pengerjaan, dan tanggal) tersimpan dalam format CSV otomatis. Guru dapat mendownload hasil kuis Tambahkan tombol “Download Hasil” (hanya muncul di mode guru atau setelah semua siswa selesai). File hasil disimpan dalam format hasil_kuis.csv berisi kolom: Nama | Skor | Waktu Selesai | Tanggal Desain Tampilan Gunakan warna edukatif modern (biru-putih lembut). Font bersih seperti Poppins atau Inter. Tata letak rapi dengan card untuk setiap soal. Teknis Simpan data hasil kuis menggunakan localStorage agar tetap tersimpan di browser pengguna. Untuk simulasi unduhan hasil, gunakan fitur ekspor CSV otomatis di JavaScript. Tidak perlu backend (cukup berbasis HTML + JS). 🧠 Contoh Tema “Kuis Perkembangan wilayah, Revolusi industri dan pengaruhnya terhadap ruang muka bumi dan kesejahteraan” (Kamu bisa isi dengan soal sesuai materi lain juga.) 💡 Tambahan (opsional) Tambahkan input nama siswa sebelum kuis dimulai. Beri animasi lembut saat berpindah soal. Tampilkan pesan motivasi di akhir, seperti: “Kerja bagus! Terus tingkatkan kemampuanmu 💪”
